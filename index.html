PokeAPI

Trabajé en una Pokédex utilizando Java como lenguaje de programación, que muestra las imágenes y características de cada Pokémon. Para obtener los datos, utilicé una PokeAPI disponible en Internet, lo que me permitió integrar información actualizada y dinámica de los Pokémon en la aplicación.

Breakout Game

Github: Breakout

Trabajé en un emocionante juego de Breakout, desarrollado completamente en Java. La idea era recrear la clásica dinámica de romper bloques con una pelota. Se implementó efectos visuales, control fluido de la paleta y una jugabilidad desafiante.

Face Detection

Trabajé con mi equipo en un proyecto para crear un sistema de control de acceso usando reconocimiento facial, con herramientas como Node-RED y DeepStack. Básicamente, este sistema permite identificar a las personas y decidir si pueden entrar o no a ciertos lugares, como oficinas o edificios. El sistema fue bastante eficiente, con un 80% de precisión, aunque siempre es importante tener en cuenta los aspectos de privacidad y consentimiento cuando se usan tecnologías de vigilancia.

Reporte: Reporte DeepStack_FaceDetection

MSQL/NODE-RED/TELEGRAM

En este proyecto, trabajé en la creación de un bot de Telegram que se conecta a una base de datos usando Node-RED. La idea principal era permitir que, a través de mensajes en Telegram, el bot pudiera interactuar directamente con la base de datos. Esto significa que podías consultar, agregar o modificar información solo enviando comandos al bot.

Reporte: Reporte MYSQL/NODE-RED/TELEGRAM
